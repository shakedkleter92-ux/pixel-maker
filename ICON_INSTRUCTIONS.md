# איך שהאייקון יופיע ב"הוסף למסך הבית"

כדי שכשמישהו נכנס לקישור ומוסיף למסך הבית יופיע האייקון שהכנת (ולא אייקון ברירת מחדל), צריך שני קבצי PNG באותה תיקייה כמו `index.html`:

- **icon-192.png** — גודל 192×192 פיקסלים  
- **icon-512.png** — גודל 512×512 פיקסלים  

## איך ליצור את הקבצים

1. **מהאייקון שבפרויקט (icon.svg)**  
   - פתחי את `icon.svg` בדפדפן (לחיצה כפולה על הקובץ).  
   - השתמשי בממיר אונליין, למשל:  
     - [cloudconvert.com/svg-to-png](https://cloudconvert.com/svg-to-png) — מעלים את ה-SVG, בוחרים גודל 192 ואז 512, ומורידים.  
     - או [svgtopng.com](https://www.svgtopng.com/) — מעלים SVG ומורידים PNG בגדלים שונים.  
   - שמרי את הקבצים בשמות **icon-192.png** ו-**icon-512.png**.  
   - העתיקי אותם **לתיקייה שבה נמצא index.html** (אותה תיקייה שמועלית ל-Netlify).

2. **להעלות מחדש ל-Netlify**  
   - וודאי שהתיקייה שמועלית כוללת: `index.html`, `manifest.json`, `icon-192.png`, `icon-512.png`.  
   - ערכי Deploy (או דחיפה ל-Git אם הפרויקט מחובר).

אחרי זה, כשמישהו נכנס ל־**https://pixelmakergrid.netlify.app** ובמובייל בוחר:
- **Android (Chrome):** תפריט ⋮ → "הוסף למסך הבית" / "התקן אפליקציה"  
- **iPhone (Safari):** כפתור "שתף" (הריבוע עם החץ) → "הוסף למסך הבית"  

האייקון שהכנת (icon-192 / icon-512) יופיע על מסך הבית עם השם "Pixel Maker".
